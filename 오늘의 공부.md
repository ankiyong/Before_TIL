# 몰랐던것

1. print() 의 sep연산자

   - print('a','b',sep='!')
     => a!b

     - 문자열만 사용이 가능하다

2. 다양한 수 표현

   - print('%x'%n)

     => n을 16진수로 나타냄

   - print('%o'%n)

     => n을 8진수로 나타냄

   - int(n,16)

     => n을 16진수로 변환

3. ord,chr

   - ord(a)

     => a문자에 해당하는 유니코드 값을 반환

   - chr(i)

     => 유니코드 i에 해당되는 문자열을 반환

4. bool(i)

   - i가 0이면 False, 이외의 수일때 True를 반환

5. Series 인덱스 순서 설정

6. pd.data_range
7. 행합계 열합계
   - df.sum() =>열 방향 합계 반환
   - df.sum(axis=1) => 행 방향 합계 반환
