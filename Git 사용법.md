# Git 사용법

 1. 파일 오픈 후 git init 입력

 2. ID,Email작성

    => git config --global user.name"이름"

    ​     git config --global user.email"이메일"

	3. 파일을 스테이징 한다

    => git add .

	4. 버전의 이름을 정한다

    => git commit -m "이름"
